# Battle Squares ⚔️

A grid-based tactical combat game where tanks, boats, and planes battle for dominance!

## Features
- Turn-based vehicle movement and attacks  
- Randomized hit accuracy for each shot  
- Terrain-specific vehicle rules (land, water, dock)
- Persistent game state between sessions

## Tech Stack
- Django (Python backend)
- HTML, CSS, JavaScript frontend

## Setup
```bash
git clone https://github.com/rileyisom/Battle-Squares.git
cd Battle-Squares
python manage.py runserver
